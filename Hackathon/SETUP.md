## Troubleshooting

**Clippy isn't responding intelligently?**
- Make sure you've entered a valid Gemini API key in settings
- Check your internet connection
- Without an API key, Clippy uses basic pattern matching

**Context awareness not working?**
- Enable "Context-Aware Help" in settings
- Make sure the app has permission to monitor active windows

**Hotkey not working?**
- Check if another app is using the same shortcut
- Change the hotkey in settings
