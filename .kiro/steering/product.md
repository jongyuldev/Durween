# Aduitor - Product Overview

Aduitor is a retro-styled AI desktop companion built as an Electron app with a floating "Ghost" character interface.

## Core Features
- Interactive ghost character that floats on desktop (draggable, always-on-top)
- AI-powered chat using Google Gemini API with multiple modes (Chat, Fast, Thinking, Search, Maps, Image Gen, Analysis)
- Task management with natural language input, categories, priorities, due dates, and reminders
- Voice interaction with real-time audio visualization
- Image generation and vision/video analysis
- Productivity tracking with streaks and stats
- Proactive assistance with contextual nudges

## User Experience
- Frameless, transparent window for seamless desktop integration
- Dark/light mode support
- Retro aesthetic with modern functionality
- Speech bubbles and animated character states (Idle, Thinking, Writing, Surprised, Listening)
