# Adiutor - Product Overview

Adiutor is a retro-styled AI desktop companion built as an Electron app. It appears as a draggable "Ghost" character that floats on the user's screen.

## Core Features

- **AI Chat**: Powered by Google Gemini API with multiple modes (Chat, Fast, Thinking, Search, Maps, Image Gen, Analysis)
- **Task Management**: Natural language task creation with categories, priorities, due dates, reminders, and tags
- **Voice Interaction**: Speech-to-text via microphone with real-time audio visualization
- **Image Generation**: Create images directly from chat using Gemini
- **Vision Analysis**: Analyze images and videos via drag-and-drop
- **Grounded Search**: Google Search and Maps integration for real-time information
- **Productivity Tracking**: Streak system with daily/weekly goals and stats dashboard
- **Proactive Assistance**: AI-driven nudges and encouragement based on task progress

## User Experience

- Frameless, transparent, always-on-top window
- Draggable ghost character with animated states (Idle, Thinking, Writing, Surprised, Listening)
- Dark/light mode support
- Retro aesthetic with modern functionality
